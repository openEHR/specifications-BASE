2+^h|*__BMM_SCHEMA_DESCRIPTOR (abstract)__*

2+a|Descriptor for a BMM schema. Contains a meta-data table of attributes obtained from a mini-ODIN parse of the  schema file.

2+|Inherits: xref:{class_bmm_definitions}[BMM_DEFINITIONS]

2+h|*Constants*

|*Bmm_internal_version*: `xref:{class_string}[String]{nbsp}={nbsp}` [1..1]
a|Current internal version of BMM meta-model, used to determine if a given schema can be processed by a given implementation of the model. +
_Inherited from xref:{class_bmm_definitions}[BMM_DEFINITIONS]_

|*Schema_name_delimiter*: `xref:{class_string}[String]{nbsp}={nbsp}"::"` [1..1]
a|Delimiter used to separate schema id from package path in a fully qualified path. +
_Inherited from xref:{class_bmm_definitions}[BMM_DEFINITIONS]_

|*Package_name_delimiter*: `xref:{class_string}[String]{nbsp}={nbsp}"."` [1..1]
a|Delimiter used to separate package names in a package path. +
_Inherited from xref:{class_bmm_definitions}[BMM_DEFINITIONS]_

|*Bmm_schema_file_extension*: `xref:{class_string}[String]{nbsp}={nbsp}".bmm"` [1..1]
a|Extension used for BMM files. +
_Inherited from xref:{class_bmm_definitions}[BMM_DEFINITIONS]_

2+h|*Attributes*

|*bmm_schema*: `xref:{class_bmm_schema}[BMM_SCHEMA]` [0..1]
a|Persistent form of model.

|*bmm_model*: `xref:{class_bmm_model}[BMM_MODEL]` [0..1]
a|Computable form of model.

|*schema_id*: `xref:{class_string}[String]` [1..1]
a|Schema id, formed from:

* `meta_data.item(Metadata_model_publisher)` '-' `meta_data.item(metadata_schema_name)` '-' `meta_data.item(Metadata_model_release)`

e.g. `openehr_rm_1.0.3`, `openehr_test_1.0.1`, `iso_13606_1_2008_2.1.2`.

|*meta_data*: `xref:{class_hash}[Hash]<xref:{class_string}[String],xref:{class_string}[String]>` [1..1]
a|Table of `{key, value}` pairs of schema meta-data, keys are string values corresponding to values in `P_BMM_SCHEMA_METADATA_KEY`.

|*includes*: `xref:{class_list}[List]<xref:{class_string}[String]>` [0..1]
a|Identifiers of schemas included by this schema.

2+h|*Functions*

|*is_top_level* (): `xref:{class_boolean}[Boolean]` [1..1]
a|True if this is a top-level schema, i.e. not included by some other schema.

|*is_bmm_compatible* (): `xref:{class_boolean}[Boolean]` [1..1]
a|True if the BMM version found in the schema (or assumed, if none) is compatible with that in this software.

|*load* (): `void` [1..1]
a|Load schema into in-memory form, i.e. a `P_BMM_SCHEMA` instance, if structurally valid. If successful, `_p_schema_` will be set.

|*validate_merged* (): `void` [1..1]
a|Validate loaded schema and report errors.

|*validate_includes* ( +
all_schemas_list: `xref:{class_list}[List]<xref:{class_string}[String]>` +
): `void` [1..1]
a|Validate includes list for this schema, to see if each mentioned schema exists in `_all_schemas_` list.

|*create_model* (): `void` [1..1]
a|Create `schema`, i.e. the `BMM_MODEL` from one `P_BMM_SCHEMA` schema.
|===
