2+^h|*TRANSLATION_DETAILS*

2+a|Class providing details of a natural language translation.

2+h|*Attributes*

|*language*: `xref:BASE:base_types:terminology_package.adoc#_terminology_code_class[Terminology_code]` [1..1]
a|Language of the translation, coded using ISO 639-1 (2 character) language codes.

|*author*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [1..1]
a|Translator name and other demographic details.

|*accreditaton*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|Accreditation of translator, usually a national translator's registration or association membership id.

|*other_details*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [0..1]
a|Any other meta-data.

|*version_last_translated*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|Version of this resource last time it was translated into the language represented by this TRANSLATION_DETAILS object.
