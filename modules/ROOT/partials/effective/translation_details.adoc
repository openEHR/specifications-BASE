2+^h|*TRANSLATION_DETAILS*

2+a|Class providing details of a natural language translation.

2+h|*Attributes*

|*language*: `xref:BASE:foundation_types:terminology.adoc#_terminology_code_class[Terminology_code]` [1..1]
a|Language of the translation, encoded following the RFC 5646 standard, using ISO 639-1 (two-character) language codes, e.g. "en". It may include a region subtag conforming to ISO 3166-1 alpha-2 (two-character country codes), e.g. "pt-br" for Brazilian Portuguese.

|*author*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [1..1]
a|Primary translator name and other demographic details.

|*accreditation*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|Accreditation of primary translator or group, usually a national translator's registration or association membership id.

|*other_details*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [0..1]
a|Any other meta-data.

|*version_last_translated*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]` [0..1]
a|Version of this resource last time it was translated into the language represented by this `TRANSLATION_DETAILS` object.

|*other_contributors*: `xref:BASE:foundation_types:structure_types.adoc#_list_class[List]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>` [0..1]
a|Additional contributors to this translation, each listed in the preferred format of the relevant organisation for the artefacts in question. A typical default is `"name <email>"` if nothing else is specified.
