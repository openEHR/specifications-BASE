2+^h|*P_BMM_PACKAGE*

2+a|Persisted form of a package as a tree structure whose nodes can contain more packages and/or classes.

2+|Inherits: xref:{class_p_bmm_package_container}[P_BMM_PACKAGE_CONTAINER], xref:{class_p_bmm_model_element}[P_BMM_MODEL_ELEMENT]

2+h|*Attributes*

|*packages*: `xref:{class_hash}[Hash]<xref:{class_string}[String],xref:{class_p_bmm_package}[P_BMM_PACKAGE]>` [1..1]
a|Package structure as a hierarchy of packages each potentially containing names of classes in that package in the original model. +
_Inherited from xref:{class_p_bmm_package_container}[P_BMM_PACKAGE_CONTAINER]_

|*documentation*: `xref:{class_string}[String]` [0..1]
a|Optional documentation of this element. +
_Inherited from xref:{class_p_bmm_model_element}[P_BMM_MODEL_ELEMENT]_

|*name*: `xref:{class_string}[String]` [1..1]
a|Name of the package from schema; this name may be qualified if it is a top-level package within the schema, or unqualified. Persistent attribute.

|*classes*: `xref:{class_list}[List]<xref:{class_string}[String]>` [0..1]
a|List of classes in this package. Persistent attribute.

|*bmm_package_definition*: `xref:{class_bmm_package}[BMM_PACKAGE]` [0..1]
a|`BMM_PACKAGE` created by `_create_bmm_package_definition_`.

2+h|*Functions*

|*merge* ( +
other: `xref:{class_p_bmm_package}[P_BMM_PACKAGE][1]` +
): `void` [1..1]
a|Merge packages and classes from other (from an included `P_BMM_SCHEMA`) into this package.

|*create_bmm_package_definition* (): `void` [1..1]
a|Generate a `BMM_PACKAGE_DEFINITION` object and write it to `_bmm_package_definition_`.
