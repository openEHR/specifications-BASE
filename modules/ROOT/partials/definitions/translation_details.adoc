h|*Class*
2+^h|*TRANSLATION_DETAILS*

h|*Description*
2+a|Class providing details of a natural language translation.

h|*Attributes*
^h|*Signature*
^h|*Meaning*

h|*1..1*
|*language*: `xref:BASE:foundation_types:terminology.adoc#_terminology_code_class[Terminology_code]`
a|Language of the translation, encoded following the RFC 5646 standard, using ISO 639-1 (two-character) language codes, e.g. "en". It may include a region subtag conforming to ISO 3166-1 alpha-2 (two-character country codes), e.g. "pt-br" for Brazilian Portuguese.

h|*1..1*
|*author*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>`
a|Primary translator name and other demographic details.

h|*0..1*
|*accreditation*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]`
a|Accreditation of primary translator or group, usually a national translator's registration or association membership id.

h|*0..1*
|*other_details*: `xref:BASE:foundation_types:structure_types.adoc#_hash_class[Hash]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String],xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>`
a|Any other meta-data.

h|*0..1*
|*version_last_translated*: `xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]`
a|Version of this resource last time it was translated into the language represented by this `TRANSLATION_DETAILS` object.

h|*0..1*
|*other_contributors*: `xref:BASE:foundation_types:structure_types.adoc#_list_class[List]<xref:BASE:foundation_types:primitive_types.adoc#_string_class[String]>`
a|Additional contributors to this translation, each listed in the preferred format of the relevant organisation for the artefacts in question. A typical default is `"name <email>"` if nothing else is specified.
